# command-line-dictionary
It is a command line dictionary tool where you get results from wordnik api

To run this tool

git clone <repo\>

cd <repo\>

npm install

chmod +x dict

// All set, You can play with this tool

./dict <word\>

./dict def <word\>

./dict ant <word\>

./dict syn <word\>

./dict ex <word\>

./dict dict <word\>
